### Description
*MAList* is an iOS app based on __VIPER (View Interactor Presenter Entity Router)__ software architectural pattern.

### Run Requirements
* Xcode 10.5 (Tested)
* Swift 5.0

### Minimum iOS Deployment Target
* iOS 10.0

# Setup
* Run ```pod install```
* Run ```open MAList.xcworkspace```
